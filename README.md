# 🌐 Simple Profile Website

A lightweight personal profile webpage built with **HTML**, **CSS**, and **JavaScript**.

---

## 📁 Project Structure
```
project/
│── index.html
│── style.css
└── main.js
```

---

## 🚀 Features
- Clean responsive design  
- Profile header (name + bio)  
- Smooth UI interactions  
- Easy to customize  
- Lightweight files  

---

## 🧩 How to Use
1. Clone the project:
```
git clone https://github.com/deveni0/profile-yousfv.git
```

2. Open the folder:
```
cd profile-yousfv
```

3. Run the site:
```
index.html
```

---

## 🎨 Customize
- Edit profile details → `index.html`  
- Change colors & fonts → `style.css`  
- Add interactions → `main.js`  

---
